# 辩境 - AI 辩论系统 使用指南
## 发给别人后，对方按下面步骤操作即可

============================================================

## 第 1 步：安装 Python（如果电脑里没有的话）

什么是 Python？
  Python 是运行这个辩论系统的"发动机"。没有它，代码跑不起来。

操作：
  1. 打开浏览器，访问 https://www.python.org/downloads/
  2. 页面中间有个大大的黄色按钮 "Download Python 3.10+"，点击下载
     （如果显示的是 3.12、3.13 等更高的版本号也没问题）
  3. 下载完成后，双击安装文件 python-xxx.exe
  4. 【重要】在安装界面底部，勾选 ☑ "Add Python to PATH"
     （这一行在安装界面最下面，默认没勾，必须手动勾上）
  5. 点击 "Install Now"（立即安装）
  6. 等进度条走完，显示 "Setup was successful" 就装好了
  7. 点击 "Close" 关闭

验证是否装成功：
  1. 按键盘 Win + R → 输入 cmd → 回车
  2. 在弹出的黑窗口里输入：python --version 然后回车
  3. 如果显示 Python 3.10.x 或更高版本，说明装好了

============================================================

## 第 2 步：解压项目文件

你（发文件的人）会把整个项目文件夹压缩成一个 zip 包发过去。

操作：
  1. 把收到的 zip 文件复制到桌面（或任何方便的位置）
  2. 右键 → "解压到当前文件夹" 或 "提取所有文件"
  3. 解压后会得到一个文件夹，名叫 "期末作业" 或你发的名字
  4. 打开这个文件夹，里面应该能看到这些文件：
     - app.py
     - template.html
     - setup.bat
     - start.bat
     - requirements.txt
     - 以及其他 .py 文件

============================================================

## 第 3 步：双击 setup.bat 安装依赖

这个脚本会自动下载和安装辩论系统需要的"零件"。

操作：
  1. 在项目文件夹里，找到 setup.bat
  2. 双击运行它
  3. 可能弹出 Windows 保护提示 "Windows 已保护你的电脑"
     → 点击 "更多信息" → 然后点击 "仍要运行"
  4. 弹出一个黑窗口，显示辩境的安装界面

在黑窗口里你会看到：

    ============================================
       辩境 - AI 辩论系统 安装程序
    ============================================

    你需要 DeepSeek API Key 才能使用 AI 辩论功能。
    如果还没有，请先申请：https://platform.deepseek.com/

    [1] 精简安装（核心功能，约 2 分钟）
    [2] 完整安装（含面部/语音分析，约 15 分钟）

    请选择 [1/2]（默认 1）：

  5. 输入 1 然后回车（推荐选精简安装）
     选 1 的话，辩论、打分、报告、摄像头、语音输入全部能用
     只有"面部表情识别"和"语音深度分析"不能用，其他一样

  6. 等待自动安装，黑窗口里的字在滚动是正常的
     精简安装大概 2 分钟，完整安装大概 15 分钟
     （安装过程中电脑可能有点卡，正常现象）

  7. 安装完成会显示：

    ============================================
        安装完成！

      下一步：
        1. 在项目文件夹新建 .env 文件
           写入：DEEPSEEK_API_KEY=你的key
        2. 双击 start.bat 启动
        3. 浏览器打开 http://127.0.0.1:5000
    ============================================

  8. 按任意键关掉这个窗口

============================================================

## 第 4 步：填写 DeepSeek API Key

辩论系统需要连接 DeepSeek 的 AI 服务才能和你辩论。
你需要一个"钥匙"（API Key）来连接。

如果你还没有 API Key：
  1. 打开 https://platform.deepseek.com/
  2. 注册账号（手机号或邮箱）
  3. 登录后点左侧 "API Keys"
  4. 点 "Create API Key" → 随便起个名字 → 复制生成的 key
  5. 新用户一般有免费额度，够用很久

操作：
  1. 在项目文件夹里，找到 .env 文件（如果没有，自己新建一个文本文档，改名为 .env）
  2. 用记事本打开 .env
  3. 看到里面有一行：DEEPSEEK_API_KEY=你的DeepSeek密钥
  4. 把"你的DeepSeek密钥"替换成你复制的 key
     例如：DEEPSEEK_API_KEY=sk-abc123def456...
  5. 保存文件并关闭

注意：
  - 等号前后不要有空格
  - 不要用中文引号
  - 粘贴后看起来应该是：DEEPSEEK_API_KEY=sk-xxxxxxxxxx

============================================================

## 第 5 步：双击 start.bat 启动

操作：
  1. 在项目文件夹里，找到 start.bat
  2. 双击运行
  3. 可能又弹出 Windows 保护提示，同样点 "更多信息" → "仍要运行"
  4. 会弹出黑窗口，显示：

    ============================================
       辩境 - AI 辩论系统 启动中...
    ============================================

    [信息] 启动辩论系统...
    [信息] 浏览器访问: http://127.0.0.1:5000

  5. 等 2-3 秒，还会出现一行：
     辩境启动 http://127.0.0.1:5000

  6. 【重要】这个黑窗口不能关！关了网站就停了
     可以把它最小化到任务栏

============================================================

## 第 6 步：打开浏览器使用

操作：
  1. 打开 Chrome、Edge 或其他浏览器
  2. 在地址栏输入：http://127.0.0.1:5000
  3. 按回车，就能看到辩论系统界面了

使用流程：
  1. 注册辩手名 → 填写名字和密码 → 点击"注册辩手"
  2. 注册成功会自动登录，进入辩论界面
  3. 点击"打开摄像头"（可选）
  4. 选择辩题（5个话题选一个）
  5. 选择你的立场（正方 / 反方）
  6. 点击"开始辩论" → AI 会先发言
  7. 在输入框打字（或点击麦克风说话）→ 点击"发送"
  8. AI 会回复你，进入你来我往的辩论
  9. 至少辩论 5 轮后，"结束辩论"按钮才会亮起
  10. 点击"结束辩论" → 系统自动生成打分报告
  11. 报告页面显示：
      - DBTI 人格类型（你的辩论性格）
      - 五维雷达图（进攻性、逻辑密度等）
      - 总分和统计数据
  12. 可以"导出报告"或查看"历史记录"

## 常见问题

Q: 双击 setup.bat 后黑窗口一闪而过？
A: 说明没有装 Python。先装 Python 再试。

Q: 安装时下载很慢？
A: 精简安装很小（几 MB），如果慢可以试试完整安装的国内镜像。
   或者检查网络连接。

Q: 打开 http://127.0.0.1:5000 显示"无法访问"？
A: 检查 start.bat 的黑窗口是否还在运行。如果关了，重新双击 start.bat。

Q: 注册时提示"网络错误"？
A: 先检查 start.bat 是否在运行。如果在运行，检查 .env 里的 API Key 是否正确。

Q: 辩论时 AI 不回复？
A: 检查 .env 文件里的 API Key 是否正确，以及 DeepSeek 账号是否有余额。

Q: 如何关闭系统？
A: 关掉 start.bat 的那个黑窗口就行，或者在黑窗口里按 Ctrl+C。

## 完整安装详细说明

完整安装比精简安装多了 3 个额外功能：
  1. 面部表情识别 — 辩论时摄像头捕捉你的表情，分析情绪变化
  2. 语音深度分析 — 语速检测、音量峰值、停顿频率、基频分析
  3. 音频处理 — 降噪、格式转换、特征提取

### 完整安装到底在装什么？

完整安装会依次安装这些大组件：

  第 1 批：PyTorch（约 300MB）
    - 这是 AI 计算框架，funasr 和 SenseVoiceSmall 需要它
    - 安装命令：pip install torch torchvision torchaudio
    - 大约 5 分钟

  第 2 批：TensorFlow + DeepFace（约 500MB）
    - TensorFlow 是谷歌的 AI 框架，DeepFace 基于它做面部识别
    - 安装命令：pip install deepface
    - deepface 会自动下载 tensorflow、opencv、keras 等
    - 注意：这一步是完整安装中最慢的，大约 8 分钟
    - 如果下载慢，脚本会自动切换到国内镜像

  第 3 批：librosa + soundfile + funasr（约 100MB）
    - librosa 做音频分析（语速、音量、停顿检测）
    - funasr 做语音识别和情绪识别
    - 这一步比较快，2-3 分钟

  以上总共约 3GB 下载，安装后约 8GB 磁盘空间。

### 常见问题

Q: 安装到一半卡住了怎么办？
A: setup.bat 支持断点续传，直接关掉重来就行。
   已经装好的包不会重复下载。

Q: 安装速度很慢 / 下载失败？
A: 脚本会自动切换到国内镜像（阿里云）重试。
   如果还是慢，可以先选精简安装用起来，
   以后再单独装需要的包。

Q: 安装后说"找不到 torch"之类的错误？
A: 重新运行 setup.bat，选 2 完整安装。
   已下载的包不会重复下载，只补装缺失的。

Q: 完整安装能不能只装一部分？
A: 可以。比如只装面部识别不要语音分析：
   打开 cmd 进入项目目录，然后执行：
   call venv\Scripts\activate.bat
   pip install deepface opencv-python
   然后直接 start.bat 就能启动。

Q: 完整安装后能不能卸载？
A: 不需要卸载，直接删掉 venv 文件夹就行。
   下次想用精简模式重新运行 setup.bat 选 1。

## 补充：下载的文件存到哪里了

当你运行 setup.bat 选完整安装时，所有下载的文件都保存在项目文件夹内部，完全不占用 C 盘：

  项目文件夹（你解压的位置）
  +-- app.py
  +-- setup.bat
  +-- start.bat
  +-- .env
  +-- venv         ← 安装的包在这里（几百 MB ~ 几 GB）
  |   +-- Libsite-packages  ← flask、openai、torch 等都在这里
  |   +-- Scripts
  +-- .pip_cache    ← 下载缓存（安装完可以删除）

- 安装完成后，.pip_cache 文件夹可以删除（释放约 1-2GB）
- venv 文件夹不能删（删了系统跑不起来）
- 想完全卸载辩论系统，直接删掉整个项目文件夹就行
